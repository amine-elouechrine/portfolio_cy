NachOS Enhanced Shell

Shell interactif avancé avec 20+ commandes intégrées, protocole FTP, et communication inter-machines

Durée:2 mois
Organisation:Ensimag
Rôle:Développeur Principal
C/C++ Shell Development System Calls File System Network Protocol FTP

Le Défi

Transformer NachOS, un système d'exploitation pédagogique minimaliste, en un environnement complet avec un shell interactif avancé intégrant gestion de fichiers, communication réseau, et exécution de commandes natives.

Le défi principal était de concevoir une architecture permettant d'exécuter des commandes directement depuis le shell (comme ls, cat, mkdir) sans nécessiter de programmes externes, tout en supportant la communication inter-machines.

  • Implémenter 35+ appels système (fichiers, processus, réseau)
  • Créer un shell avec 20+ commandes intégrées
  • Développer un protocole FTP pour transfert de fichiers entre machines NachOS
  • Garantir zéro fuite mémoire (validation Valgrind)

Mon Rôle

Développeur système principal, responsable de la conception et de l'implémentation complète du shell interactif et de ses fonctionnalités :

  • Shell Enhanced : parseur de commandes, interpréteur, gestion des arguments
  • Commandes système de fichiers : ls, cat, mkdir, cd, pwd, write, create
  • Commandes réseau : send/recv, protocole FTP (get/put/ls/rm)
  • Intégration noyau : liaison shell ↔ appels système
  • Script d'automation : launch-shell.sh pour compilation et lancement rapide

La Solution

Shell Interactif Complet

Développement d'un shell intégré au noyau permettant l'exécution de commandes natives sans programmes externes. Architecture modulaire avec parseur intelligent, gestionnaire de commandes, et interface unifiée kernel/user.

Fonctionnalités Implémentées

Commandes Système Fichiers (7)

  • ls [path] - Listage de répertoires
  • cat <file> - Affichage de fichiers
  • mkdir <dir> - Création de répertoires
  • cd <path> - Changement de répertoire
  • pwd - Affichage du répertoire courant
  • create <file> - Création de fichiers
  • write <file> <text> - Écriture dans fichiers

Commandes Réseau FTP (7)

  • send <to> <msg> - Envoi de messages
  • recv - Réception de messages (bloquant)
  • ftp-get <m> <remote> <local> - Téléchargement de fichiers
  • ftp-put <m> <local> <remote> - Upload de fichiers
  • ftp-ls <m> [path] - Listage distant
  • ftp-mkdir <m> <path> - Création répertoire distant
  • ftp-rm <m> <file> - Suppression distante

Commandes Système (6)

  • help - Affichage de toutes les commandes
  • exec <program> - Exécution de programmes externes
  • echo <text> - Affichage de texte
  • clear - Effacement de l'écran
  • quit / exit - Fermeture du shell
  • halt - Arrêt du système

Architecture Technique

  • Parseur de commandes : support de 10 arguments max, buffer 256 caractères
  • Protocole FTP custom : transfert inter-machines NachOS via réseau
  • 3 Flavors NachOS : step4 (syscalls de base), step5 (filesystem), network (complet)
  • Script d'automation : launch-shell.sh - compilation, formatage, lancement en une ligne
  • Tests multi-machines : support de communication entre 2+ instances NachOS
